# Profitability-Analysis

![](Furniture.avif)
## Introduction
This is a Sales Analysis of an imaginary store called **KI Store** using Power BI. The project's intent is to analyze the store historical sales data and derive key insights on profitability and strategy.

**_Disclaimer_**: _All datasets and reports do not represent any company, institution or country, but just a dummy dataset to demonstrate Power BI capabilities._

## Problem Statement
1. Has Profitablity decreased in the last year?
2. Which products have the highest impact on revenue and profits in the past year?
3. How should strategy be updated for the upcoming year to improve profitability?

## Skills/Concepts demonstrated
* DAX
* Quick Measures
* Modeling (Star Schema)
* Version Control

## Modeling 
The model is a star schema 
There are one fact table and one dimension table (generated). The dimension table is joined to the fact table with a one-to-many relationship.

## Visualization 
The report comprises 2 pages
1. Profitability by Year and State
2. Profitability Analysis
